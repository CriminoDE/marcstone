# Vision Phase 2 + 3 - Optik, Atmosphaere, Sound

Sammlung aller kreativen Ideen fuer das Rebranding (Phase 2) und den Juice (Phase 3).
Henrys Ansage: "einfach ein geileres Hearthstone." Darf nicht verloren gehen.

## Stil / Setting
- Dunkel, mystisch, mittelalterlich, nordisch/Wikinger + Seher/Hexen/Wahrsager-Ton.
- Modern, schick, krass. Gerne **3D**-Elemente.
- Tarot-Karten-Aesthetik als Einfluss (Marc wollte immer ein Tarot-Karten-Ding bauen) -> Karten koennten Tarot-Anmutung haben (Rahmen, Symbolik, Mystik).
- Tools: **frontend-design-Skill** + **21st.dev** (UI-Elemente + Effekte) sind gesetzt. Dazu Eagle-Library + Pexels/Pixabay fuer Artwork/Hintergruende.

## Atmosphaere / Ambient (Phase 3)
- **Raben** fliegen ueber den Bildschirm + Raben-Geraeusche (Krächzen). Passt nordisch (Odins Raben Hugin & Munin).
- **Wasser-Rauschen** im Hintergrund.
- **Woelfe** (Geheul / Knurren) - Fenrir-Vibe.
- **Dumpfe Krieger** / dumpfe Kriegstrommeln im Hintergrund.
- Leise Mittelalter-Hintergrundmusik.

## Timer-Effekt (Phase 3)
- Statt nur Zahl + Ticken: eine **brennende Zuendschnur** die runterbrennt, wenn die Zugzeit ablaeuft. Visuell spuerbar dass die Zeit knapp wird.

## Marc-Themen-Karten (maechtige Legendaere)
Eigene, witzig-epische Karten rund um Marc:
- **Marcs Fluch** (Marc's Curse)
- **Zorn des Marc** (Wrath of Marc)
- **Marksmann** (Wortspiel marksman / Marc-mann) - z.B. ein praeziser Schuetzen-Diener
- Weitere in dem Stil ergaenzen (Marcs Bann, Marc der Seher, Marcs Raben, ...).

## Spielname - Kandidaten
Aktueller Arbeitstitel: Marcstone. Finaler Name wird in Phase 2 gelockt.

Engere Auswahl bisher:
- **Marchalla** (Valhalla + Marc) - Empfehlung
- **Marcgard** (Asgard/Midgard-Reich)
- **Jarl Marc** (Wikinger-Fuerst, koeniglich)
- **Marcnarök** (Ragnarök)
- **Der Markseher** (Seher/Hexen-Richtung)

Weitere im Rennen (Wortspiel + nordisch):
- **Marcwald** (Wald des Marc + Germ. Herrscher-Endung -wald)
- **Die Raben des Marc / Marcraben** (greift die Raben-Atmosphaere auf)
- **Ragnar Marc** (Ragnar-Lothbrok-Vibe)
- **Marcborg** (borg = nordische Festung/Burg)
- **Helmarc** (Hel = nordische Unterwelt, dunkel/hexig)
- **Marctyr** (Tyr = Kriegsgott, klingt wie Martyr)
